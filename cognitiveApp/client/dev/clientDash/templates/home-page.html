Cognitive 911 Chat-bot



Team Motorola looks to build an assistive cognitive-bot for 911 call-takers. Through leveraging Amazon Web Services (AWS) technologies and frameworks, we can interact with multiple callers simultaneously though voice and text. By utilizing a modified MEAN JavaScript web stack, we will develop an intuitive web application that can provide 911 call-takers with meaningful insights. In doing so, call-takers can better assist 911 callers by providing better and more informative service, thus improving and even saving lives.
